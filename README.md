# ydd_m

> Nuxt.js project

## Build Setup

``` bash
# 依赖安装
$ npm install # Or yarn install

# 在开发环境中运行
$ npm run dev

# 在测试环境部署
$ npm run build-dev
$ node server.js

# 生产环境部署
全局安装PM2 
npm i pm2 -g

# e2e 测试
先执行命令启动项目: npm run dev
在执行e2e测试命令: npm run e2e

$ npm run build
$ pm2 start server.js

# generate static project
$ npm run generate
```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
